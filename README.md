# Pills Detection Project with ML & XML - Object Detection

This project uses computer vision to detect and identify medical pills.

## 📂 Project Structure
- `pillsPicture/`: Contains images and annotations.
- `output/`: Results from the detection process.
- `.record` files: Handled via Git LFS for training data.

## 🛠️ Setup
1. Install Git LFS: `git lfs install`
2. Pull the large files: `git lfs pull`
